# Bangrims-Modlist
My current main modlist for full futuristic playthrough ^_^
I've currently noticed one issue, which prevented me from releasing the list earlier, but I found it not to be a big deal in my playthroughs. Basically normal trade caravans don't spawn, but orbital ones do. Since the orbital traders have up to 200k silver with them, I find them to be the only relevant ones in the lategame anyway. I just wanted to put in a disclaimer, enjoy!
